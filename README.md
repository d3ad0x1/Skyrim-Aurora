# Skyrim: Aurora

> A lightweight Vanilla+ modlist for Skyrim Anniversary Edition.

Aurora is designed to modernize Skyrim while preserving the original gameplay experience.

## Goals

- Beautiful visuals
- Stable gameplay
- Lore-friendly
- Modern interface
- Best community quests
- Excellent performance

## Philosophy

✔ Vanilla+

✔ Lightweight

✔ Stable

✔ Modern

✔ Performance First

✔ Quality over Quantity

## Not Included

✘ Souls-like Combat

✘ Hardcore Survival

✘ NSFW

✘ Anime

✘ Requiem
