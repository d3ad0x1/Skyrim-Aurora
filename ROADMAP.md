# Skyrim: Aurora Roadmap

> Project roadmap for Skyrim: Aurora

Current Version: **0.1.0**

Status: 🚧 In Development

---

# Vision

Aurora aims to become one of the best Vanilla+ Skyrim modlists by combining:

- Beautiful modern graphics
- Stable gameplay
- High performance
- Lore-friendly content
- Modern user experience
- Easy installation

# Project Principles

Aurora will always be:

- Vanilla+
- Lore-friendly
- Lightweight
- Stable
- Modern
- Performance First
- Quality over Quantity

Aurora will never include:

- Souls-like gameplay
- Hardcore survival
- NSFW content
- Anime-style assets
- Full gameplay overhauls
- Requiem
