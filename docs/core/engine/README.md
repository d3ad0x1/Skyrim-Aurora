# Aurora Engine

> Core engine components used by Skyrim: Aurora.

Status

🚧 Development

---

## Purpose

Aurora Engine contains the essential engine-level components required for a stable Skyrim Anniversary Edition experience.

These components provide:

- Script extender support
- Engine fixes
- Stability improvements
- Framework libraries
- Crash diagnostics

## Installation Policy

Each component is installed individually.

After every installation:

1. Launch the game.
2. Verify functionality.
3. Check logs.
4. Benchmark startup.
5. Update documentation.
